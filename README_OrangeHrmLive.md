Selenium Automation Framework

Description:
This project contains automation test script developed using Selenium and Java.
It follows Page Object Model design pattern and TestNG for execution.

Tools & Technologies used:
Java
Selenium WebDriver
TestNG
Maven
Git and GitHub

Project Structure:
src/main/java contains the common methods, config properties file, page classes, and utility classes to handle waits, test data etc.
src/test/java contains the base test class and the main test class where actual business flow is automated.
pom.xml handles the maven dependencies
testng.xml handles the test execution

How to Run the project:
Using Eclipse: Import project as Maven project and update the dependencies.
Run the test class from testng.xml file.

Key Features:
Used explicit and fluent waits
Followed POM structure
Assertions for validation
